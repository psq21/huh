## setup 
```
cd src

# activate venv (optional)
python -m venv venv
source venv/Scripts/activate # or the equivalent in your OS

# install deps
pip install -r requirements.txt

# run
python index.py
```

use ``/src`` as the cwd!

format python files: ``black .``
format html: ``djlint . --reformat``
